# Diretório de Especificações e Documentos de Negócio (specs)

Este diretório é dedicado a armazenar todas as especificações do sistema, documentações de regras de negócio, PRDs (Product Requirement Documents) e protótipos para o projeto **Amigo Secreto ou Inimigo**.

## 📝 O que colocar aqui?
*   **PRDs e Escopo do Projeto**: Definições dos objetivos, público-alvo e escopo.
*   **Regras de Negócio**: Documentação detalhada sobre o funcionamento do sorteio, distribuição de papéis (Amigo vs. Inimigo), restrições e regras de validação.
*   **Diagramas e Casos de Uso**: Fluxogramas do sorteio e diagramas de sequência das interações dos usuários.
