# Diretório de Bibliotecas Compartilhadas (libs)

Este diretório é dedicado a conter submódulos e bibliotecas compartilhadas entre o Frontend (Angular) e o Backend (NestJS).

## 🚀 O que colocar aqui?

- **Interfaces e Tipos Compartilhados**: Definições de DTOs e contratos de API comuns.
- **Utilitários**: Helpers de formatação, validação ou criptografia usados por ambos os lados.
